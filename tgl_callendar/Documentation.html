Date format & Time Format

Change date format, easily and simple

d : 1-31

dd : 01-31

ddd : mon-sun

dddd : monday-sunday

m : 1-12

mm : 01-12

mmm : jan-dec

mmmm : january-december

y : 0-99

yy : 01-99

yyy : 001-999

yyyy : 0001-9999

and this for time format

h : 0-23

hh : 00-23

n : 0-59

nn : 00-59

s : 0-59

ss : 00-59
